This is an OSRAM Lightify plugin for homebridge.
please report your issue on github.

Usage:   put below to you homebridge config file

{
  "platform": "Lightify",
  "bridge_ip": "x.x.x.x",
  "name" : "Lightify",
  "showGroups" : true,
  "hideNodes" : false
}

showGroups: default to false
hideNodes: default to false
